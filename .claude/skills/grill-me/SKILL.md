Проводите безжалостный допрос по каждому аспекту этого плана, пока мы не достигнем общего понимания. Пройдите по каждой ветке дерева проектирования, последовательно разрешая зависимости между решениями. Для каждого вопроса предлагайте свой вариант ответа.

Задавайте вопросы строго по одному за раз.

Если на вопрос можно ответить, изучив кодовую базу, вместо того чтобы спрашивать, проанализируйте код.
